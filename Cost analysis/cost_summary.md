## 💡 Топ-3 способа снизить стоимость

1. **Аудио (76% затрат)**:
   - Использовать gTTS (бесплатно) вместо OpenAI TTS
   - ⭐ **Pollinations.ai Free tier** (текущая реализация) — БЕСПЛАТНО до 200 images/час
   - Кэшировать повторяющиеся фразы
   - Уменьшить качество (битрейт)

2. **Изображения** (если используются):
   - Stable Diffusion API вместо DALL-E (дешевле в 10x)
   - Генерировать только обложку
   - Unsplash API (бесплатные стоковые фото)

3. **Текст (23% затрат)**:
   - Меньше Pinecone результатов (меньше context)
   - Убрать итерации (строже валидация)
   - GPT-3.5-turbo для простых проектов
