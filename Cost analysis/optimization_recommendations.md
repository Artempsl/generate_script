---

#### 1.3 Batch обработка для TTS
```python
# Отправлять несколько сегментов одновременно
# OpenAI позволяет batch requests
async def batch_tts_generation(segments):
    tasks = [generate_audio(seg) for seg in segments]
    return await asyncio.gather(*tasks)
```
**Экономия**: Нет прямой экономии, но быстрее на 30-40%  
**Сложность**: Средняя  
**Время внедрения**: 2-3 дня
