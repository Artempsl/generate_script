## 💰 Актуальные цены API (на апрель 2026)

### 1. OpenAI GPT-4o-mini
- **Input tokens**: $0.150 / 1M токенов  
- **Output tokens**: $0.600 / 1M токенов  
- **Соотношение input/output**: ~70% input / 30% output (на основе анализа)

### 2. OpenAI TTS (Text-to-Speech)
- **Модель**: TTS-1  
- **Цена**: $0.015 / 1,000 символов  
- **Голос**: alloy (стандартный)

### 3. Cohere Embeddings
- **Модель**: embed-english-v3.0  
- **Цена**: $0.100 / 1M токенов  
- **Использование**: ~512 токенов на запрос × 5 запросов = 2,560 токенов

### 4. Pinecone (Векторная БД)
- **План**: Serverless (pay-as-you-go)  
- **Read Units**: $0.20 / 1M запросов  
- **Хранение**: ~$0.25 / GB / месяц (не учитывается в расчете на проект)

### 5. Facticity API (Teacher Pipeline)
- **Использование**: Только в teacher pipeline  
- **Цена**: Зависит от плана (предполагаем включено)  
- **Запросов на проект**: 2-5 fact-checks

### 6. MoviePy (Видео рендеринг)
- **Стоимость**: $0 (локальный рендеринг)
- **Зависимости**: FFmpeg (бесплатный)

1. **TTS — самая дорогая часть** (77% затрат)

3. **Embeddings и Pinecone незначительны** (<1%)

## 💡 Рекомендации по оптимизации

### Снижение стоимости TTS:
1. **Использовать gTTS** (Google TTS) — бесплатно, но хуже качество
2. **Кэшировать аудио** для повторяющихся фраз
3. **Batch processing** — генерировать аудио оптом

### Снижение стоимости GPT:
1. **Уменьшить context** — меньше Pinecone результатов
2. **Меньше итераций** — строже валидация с первого раза
3. **Gpt-3.5-turbo** для простых задач (но хуже качество)

### Для изображений:
1. ⭐ **Pollinations.ai (Free tier)** — БЕСПЛАТНО до 200 images/час (текущая реализация)
2. **Stable Diffusion** вместо DALL-E (дешевле в 10x)
3. **Генерировать только обложку**, а не на каждый сегмент
4. **Использовать стоковые изображения** с Unsplash API (бесплатно)

**Текущая реализация:** Система уже использует Pollinations.ai с fallback chain и бесплатными начислениями.
