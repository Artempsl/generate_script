## 🧮 Расчет для 1-минутного видео

### Сценарий: Генерация с консистентными изображениями

**Используемая технология:**
- **generator_v2.py** — новая система с entity consistency
- 8 изображений на 1-минутное видео (1 на сегмент)
- Подход:
  1. Генерация entity reference images (Flux model)
  2. Генерация segment images с P-Image-Edit (image-to-image для консистентности)

### Breakdown по pollen

#### Вариант 1: Z-Image-Turbo (текущий fallback)
```
8 изображений × 0.002 pollen = 0.016 pollen
```

#### Вариант 2: P-Image для сегментов
```
8 изображений × 0.005 pollen = 0.04 pollen
```

#### Вариант 3: P-Image-Edit (image-to-image, консистентность)
```
2 entity references (Flux) = 2 × 0.001 = 0.002 pollen
8 segment images (P-Image-Edit) = 8 × 0.01 = 0.08 pollen
───────────────────────────────────────────────────
ИТОГО: 0.082 pollen
```

## 💵 Конвертация Pollen → USD

### Проблема: Цена Pollen не публикуется

**Что известно:**
1. Pollinations позиционируется как **бесплатная альтернатива** DALL-E/Midjourney
2. Основной источник pollen — **бесплатные начисления** (0.15-0.8/час)
3. Packs доступны, но цена индивидуальна

### Оценка на основе бесплатных начислений

**Сценарий 1: Seed tier (0.15 pollen/hour)**
```
Время на накопление 0.082 pollen:
0.082 ÷ 0.15 = 0.55 часа ≈ 33 минуты ожидания

При генерации 1 видео/день: БЕСПЛАТНО (накапливается за ночь)
```

**Сценарий 2: Flower tier (0.4 pollen/hour)**
```
Время на накопление 0.082 pollen:
0.082 ÷ 0.4 = 0.21 часа ≈ 12 минут ожидания

При генерации 3-4 видео/день: БЕСПЛАТНО
```

**Сценарий 3: Nectar tier (0.8 pollen/hour)**
```
Время на накопление 0.082 pollen:
0.082 ÷ 0.8 = 0.1 часа ≈ 6 минут ожидания

При генерации 6-8 видео/день: БЕСПЛАТНО
```

### Оценка при покупке Pollen Packs

**Гипотетическая оценка** (на основе сравнения с конкурентами):

Если 1 pollen ≈ $0.01-0.05 (консервативная оценка):
```
Вариант 1 (Z-Image-Turbo):
0.016 pollen × $0.01-0.05 = $0.00016 - $0.0008 ≈ $0.0005

Вариант 2 (P-Image):
0.04 pollen × $0.01-0.05 = $0.0004 - $0.002 ≈ $0.001

Вариант 3 (P-Image-Edit, консистентность):
0.082 pollen × $0.01-0.05 = $0.00082 - $0.0041 ≈ $0.002
```

**Сравнение с известными ценами:**
- DALL-E 3: $0.040/image × 8 = **$0.32**
- Stable Diffusion: $0.004/image × 8 = **$0.032**
- Pollinations (оценка): **$0.0005 - $0.002** ✨

## 🎯 Ключевые выводы

### ✅ Преимущества Pollinations.ai

1. **Практически бесплатно** при использовании бесплатных начислений
   - Seed tier: 75 images/hour (достаточно для 9 видео/час)
   - Flower tier: 200 images/hour (25 видео/час)
   - Nectar tier: 400 images/hour (50 видео/час)

2. **Экономия до 99.4%** по сравнению с DALL-E:
   - DALL-E: $0.32 на изображения
   - Pollinations (free): $0.00 на изображения
   - **Экономия: $0.32 = 100%**

3. **P-Image-Edit для консистентности** — специализированная модель для image-to-image:
   - Поддержка reference images для character consistency
   - Стоимость: 0.01 pollen/img (vs DALL-E $0.04/img = **75% экономия**)

4. **Fallback chain** в коде:
   ```python
   FALLBACK_MODELS = ["z-image-turbo", "p-image", "flux", "gpt-image-1-mini"]
   ```
   - Автоматический выбор наиболее доступной модели
   - Retry logic при ошибке 402 (Insufficient Balance)

### ⚠️ Ограничения

1. **Rate limits** на бесплатном tier:
   - pk_PUBLISHABLE: 1 pollen/IP/hour (только для демо)
   - sk_SECRET: нет rate limits, но ограничен балансом

2. **402 Payment Required** при недостаточном балансе:
   - Код ожидает 60 секунд (`BALANCE_WAIT = 60`)
   - Затем повторяет попытку

3. **Нет DPA** (Data Processing Agreement):
   - GDPR risk: MEDIUM
   - План миграции на Stability AI в Q3 2026 (есть DPA, $0.04/img)

4. **Зависимость от tier**:
   - Tier определяется активностью (GitHub, shipped apps, usage)
   - Новые проекты начинают с Seed tier (0.15 pollen/hr)
